# Android debloating scripts

A bunch of scripts that I have used to debloat non-rooted Android
devices.  If something breaks, reinstall the packages using

    adb shell cmd package install-existing <package_name>

Fetch/update
[uad_list.json](https://github.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation/raw/refs/heads/main/resources/assets/uad_lists.json)
(from the Universal Android Debloater project) before using the scripts.

## Alternatives

- [Universal Android Debloater](https://github.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation)
- [Canta](https://github.com/samolego/Canta)
